このサイトをAstroで作りました。
テンプレートからスタートして、3つのページ（HOME・WORKS・BLOG）を追加しています。

## Astroを選んだ理由

- MDX / Markdown でページを管理できる
- 必要なときだけJavaScriptを読み込むため軽量
- React, Vue, Svelte, SolidJSなどを一つのプロジェクト内で混ぜて使える
- GitHub Pagesで公開したときにGitHub Actionsを使うと自動でビルド・公開される仕組みを作れる

今後もこのサイトに機能を追加していく予定です。
