# スマホ表示時のリンク中央寄せ対応

ご要望の通り、スマホ表示時の「HOME, WORKS, BLOG」のリンクグループを横幅の中央に配置し、ハンバーガーメニューは右側のままとなるようレイアウトを調整しました。

## 概要

`src/components/Header.astro` 内の `nav` 要素のスタイルを Flexbox から Grid レイアウトに変更することで、美しい中央揃えを実現しました。

## 修正内容

*   **Gridレイアウトの導入**:
    *   モバイル向けの `nav` を `display: grid;` に変更し、カラム構成を `grid-template-columns: 1fr auto 1fr;` に設定しました。
    *   これにより、左右の余白（`1fr`）が常に等間隔となり、中央の列（`auto`）が画面の真ん中に完璧に配置されるようになります。
*   **各要素の配置**:
    *   リンクのグループ (`.internal-links`) を `grid-column: 2;` に指定し、中央の列に配置しました。
    *   ハンバーガーメニュー (`.hamburger`) を `grid-column: 3;` とし、さらに `justify-self: flex-end;` で右端へ寄せることで、希望通りのレイアウト構造に整えました。
*   **メニュー展開時の挙動維持**:
    *   開いた際のリンク一覧 (`.internal-links.open`) には `grid-column: 1 / -1; grid-row: 2;` を指定し、横幅いっぱいに次の行（2行目）へ展開されるよう担保しています。

## バリデーション

*   `npm run build` を実行し、問題なくビルドされることを確認しました。
*   ローカル・開発環境にて、ブラウザの表示が即時反映されている状態です。
